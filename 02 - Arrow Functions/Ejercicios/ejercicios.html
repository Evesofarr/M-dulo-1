Funciones Arrow con Arrays: map: Dada una matriz de nombres, crea una función arrow que use map para convertir todos los nombres a mayúsculas. Dada una matriz de temperaturas en grados Celsius, crea una función arrow que use map para convertirlas a grados Fahrenheit. Dada una matriz de objetos de productos, crea una función arrow que use map para obtener un arreglo de los nombres de los productos. filter: Dada una matriz de números, crea una función arrow que use filter para obtener todos los números pares. Dada una matriz de palabras, crea una función arrow que use filter para obtener todas las palabras que contienen la letra "e". Dada una matriz de edades, crea una función arrow que use filter para obtener todas las edades menores de 30. reduce: Dada una matriz de números, crea una función arrow que use reduce para calcular la suma de todos los elementos. Dada una matriz de palabras, crea una función arrow que use reduce para concatenar todas las palabras en una sola cadena. Dada una matriz de objetos de productos con precios, crea una función arrow que use reduce para calcular el precio total de todos los productos. Funciones Arrow Puras: Crea una función arrow pura que tome dos números como argumentos y devuelva su suma. Crea una función arrow pura que tome un número como argumento y devuelva su cuadrado. Crea una función arrow pura que tome un arreglo de números como argumento y devuelva el número más grande. Crea una función arrow pura que tome una cadena de texto como argumento y devuelva la longitud de la cadena. Crea una función arrow pura que tome un objeto con las propiedades ancho y alto (que representan el ancho y el alto de un rectángulo) y devuelva su área calculada (Área = ancho * alto). Crea una función arrow pura que tome un arreglo de nombres como argumento y devuelva un arreglo de nombres ordenados alfabéticamente.
